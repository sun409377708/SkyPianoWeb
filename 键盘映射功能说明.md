# 🎹 键盘映射功能说明

## 功能概述

现在可以使用电脑键盘来演奏钢琴了！按下键盘上的字母键，对应的钢琴键会发出声音。

---

## 键位布局

钢琴共有 **15个键**，分为 **3行**，每行 **5个键**：

```
┌─────────────────────────────────┐
│  钢琴键盘布局（从左到右）         │
├─────────────────────────────────┤
│  第一行：◇  ○  ◇  ○  ◇          │
│  音符：  C4 D4 E4 F4 G4          │
│  按键：  Y  U  I  O  P           │
├─────────────────────────────────┤
│  第二行：○  ◇  ◇  ◇  ○          │
│  音符：  A4 B4 C5 D5 E5          │
│  按键：  H  J  K  L  ;           │
├─────────────────────────────────┤
│  第三行：◇  ○  ◇  ○  ◇          │
│  音符：  F5 G5 A5 B5 C6          │
│  按键：  N  M  ,  .  /           │
└─────────────────────────────────┘
```

## 使用方法

### 1. 基本演奏
- 按下键盘上的字母键，钢琴会发出对应的声音
- 释放按键，声音停止
- 可以同时按下多个键（和弦）

### 2. 示例曲目

**小星星（前半段）**：
```
U U K K L L K
J J I I O O U
```

**生日快乐歌（简化版）**：
```
U U I U P O
U U I U L P
```

### 3. 练习建议
1. 先熟悉单个按键的位置
2. 练习简单的音阶：Y U I O P
3. 尝试弹奏简单的旋律
4. 配合鼠标点击使用

## 技术实现

### 代码位置
- **键盘映射逻辑**：`src/components/keyboard/Layout.vue`
- **提示组件**：`src/components/KeyboardHint.vue`

### 核心代码
```javascript
const KEY_MAP = {
  // 第一行：y u i o p
  'y': 0, 'u': 1, 'i': 2, 'o': 3, 'p': 4,
  // 第二行：h j k l ;
  'h': 5, 'j': 6, 'k': 7, 'l': 8, ';': 9,
  // 第三行：n m , . /
  'n': 10, 'm': 11, ',': 12, '.': 13, '/': 14
}
```

### 特性
- ✅ 防止重复触发（按住不会连续触发）
- ✅ 防止在输入框中触发
- ✅ 支持同时按下多个键
- ✅ 自动清理事件监听器
- ✅ 视觉反馈（按键会有按下效果）

## 自定义键位映射

如果想修改键位映射，编辑 `src/components/keyboard/Layout.vue`：

```javascript
const KEY_MAP = {
  // 修改这里的映射关系
  'a': 0,  // 将第一个键改为 A
  's': 1,  // 将第二个键改为 S
  // ...
}
```

---

## 注意事项

1. **大小写不敏感**：按 `Y` 或 `y` 效果相同
2. **输入框保护**：在输入框中输入时不会触发钢琴
3. **浏览器兼容性**：支持所有现代浏览器
4. **移动端**：键盘映射仅在桌面端有效，移动端请使用触摸

## 常见问题

**Q: 为什么选择这些键位？**  
A: 这些键位在键盘上呈现3行5列的布局，与钢琴的3x5布局对应，便于记忆和操作。

**Q: 可以同时按多个键吗？**  
A: 可以！支持同时按下多个键来演奏和弦。

**Q: 按键没有反应怎么办？**  
A: 检查是否在输入框中，或者刷新页面重试。

**Q: 能否支持更多音符？**  
A: 当前钢琴只有15个键，如需扩展可以修改 `createKeyboard` 函数的 `total` 参数。
