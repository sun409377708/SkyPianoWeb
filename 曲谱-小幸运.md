## 完整版演奏指南

### 读谱方法

1. **纵向阅读**：从上往下，每一行是一个时间点
2. **同时按键**：同一行有多个数字，表示同时按下（和弦）
3. **空行**：表示休止或延音
4. **密集的行**：表示快速演奏

### 节奏标记

- `○` = 长音（按住2拍）
- `●` = 短音（按1拍）
- `-` = 休止

---

## 练习建议

### 第一步：分段练习
1. 先练习前奏（8小节）
2. 再练习主歌A段
3. 然后练习主歌B段
4. 最后练习副歌

### 第二步：连贯演奏
- 慢速完整演奏一遍
- 找出卡顿的地方重点练习
- 逐渐提速

### 第三步：加入感情
- 注意强弱变化
- 副歌部分可以稍微用力
- 主歌部分轻柔一些

## 和弦伴奏（可选）

如果想要更丰富的效果，可以在旋律基础上加入和弦：

### C大调和弦
- **C和弦**：Y + I + P (C + E + G)
- **F和弦**：O + H + K (F + A + C)
- **G和弦**：P + J + L (G + B + D)
- **Am和弦**：H + K + ; (A + C + E)
