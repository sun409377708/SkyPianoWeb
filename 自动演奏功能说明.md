# 🎵 Sky-Piano 自动演奏功能说明

## ✨ 新功能上线！

现在可以**自动演奏**简谱了！点击左上角的🎵按钮即可打开自动演奏面板。

---

## 🎹 使用方法

### 方法1：选择预设曲目

1. 点击左上角的 **🎵** 图标打开面板
2. 选择一首预设曲目：
   - **欢乐颂** - 贝多芬经典
   - **小星星** - 儿歌经典
   - **两只老虎** - 简单易学
   - **生日快乐** - 庆祝必备
3. 点击 **▶️ 播放** 按钮
4. 享受自动演奏！

### 方法2：输入自定义简谱

1. 在文本框中输入键位序列
2. 格式：`Y Y U I I U U | I I O P P O I U`
3. 用空格分隔每个音符
4. 用 `|` 分隔小节（可选）
5. 点击 **加载自定义** 按钮
6. 点击 **▶️ 播放** 按钮

---

## 🎮 控制按钮

| 按钮 | 功能 | 说明 |
|------|------|------|
| **▶️ 播放** | 开始播放 | 从头开始或继续播放 |
| **⏸️ 暂停** | 暂停播放 | 暂停当前位置 |
| **⏹️ 停止** | 停止播放 | 停止并回到开头 |

---

## 📊 播放进度

面板会实时显示：
- **进度条** - 可视化播放进度
- **音符计数** - 当前音符/总音符数
- **百分比** - 播放完成百分比
- **当前曲目** - 正在播放的曲目名称

---

## 📝 简谱格式说明

### 键位对照

```
Y U I O P  →  C4 D4 E4 F4 G4  (Do Re Mi Fa Sol)
H J K L ;  →  A4 B4 C5 D5 E5  (La Si Do Re Mi)
N M , . /  →  F5 G5 A5 B5 C6  (Fa Sol La Si Do)
```

### 示例：欢乐颂

**简谱**：
```
3 3 4 5 | 5 4 3 2 | 1 1 2 3 | 3 2 2 -
```

**键位**：
```
I I O P | P O I U | Y Y U I | I U U -
```

**输入格式**：
```
I I O P P O I U Y Y U I I U U
```

## 🔧 高级功能

### 自定义节奏（未来功能）

目前所有音符都是相同时长（500ms）。未来版本将支持：
- 长音、短音标记
- 自定义BPM（速度）
- 休止符
- 和弦（同时按多个键）

### JSON格式（开发者）

可以创建JSON文件实现更精确的控制：

```json
{
  "title": "欢乐颂",
  "bpm": 120,
  "notes": [
    {"key": "I", "duration": 500, "note": "E4"},
    {"key": "I", "duration": 500, "note": "E4"},
    {"key": "O", "duration": 500, "note": "F4"}
  ]
}
```

## 💡 使用技巧

### 1. 学习新曲目
- 先用自动演奏听一遍旋律
- 观察琴键的按下顺序
- 暂停后手动跟着练习

### 2. 创作简谱
- 先手动弹奏，记录键位
- 整理成文本格式
- 用自动演奏验证

### 3. 分享曲谱
- 将键位序列复制出来
- 分享给朋友
- 他们可以直接粘贴播放

```javascript

  title: '我的曲目',
  keys: 'Y U I O P H J K L'
}
```

## 🐛 常见问题

**Q: 播放速度太快/太慢？**  
A: 目前速度固定为500ms/音符。未来版本会支持调速。

**Q: 能播放和弦吗？**  
A: 目前只支持单音。和弦功能正在开发中。

**Q: 能导入MIDI文件吗？**  
A: 可以！直接拖拽MIDI文件到页面即可自动播放。

**Q: 能录制我的演奏吗？**  
A: 录制功能正在开发中，敬请期待。

**Q: 能保存自定义曲目吗？**  
A: 目前需要手动保存文本。未来会支持本地存储。
